17. Color Contrast

WCAG AA requires 4.5:1 for normal text, 3:1 for large text. Check every text/background pair before shipping. Switch all 4 theme types to verify.

Text tokenBackground tokenRatio (dark)StatusNotes
--foreground--background~17:1✅ PASSPrimary text on app bg
--foreground--alt-background~11:1✅ PASSText on notifications, panels
--foreground--alt-background-neutral~11:1✅ PASSText on inputs, secondary buttons
--foreground--accent~5.6:1✅ PASSPrimary button text — use --foreground, NOT --background
--alt-foreground--background~5.8:1✅ PASSSecondary text on app bg
--alt-foreground--alt-background~4.5:1✅ PASS (borderline)Secondary text on panels
--alt-foreground-neutral--alt-background-neutral~4.1:1⚠️ LARGE ONLYUse ≥ 18px or weight ≥ 600 for this pair
--background--accent~2.6:1❌ FAILDo NOT use --background text on accent bg
⚠️ Critical: The primary button must use color: var(--foreground) — never color: var(--background). In dark mode, --background is near-black (~#262626) on medium-dark purple, which produces only ~2.6:1 contrast ratio and fails WCAG AA.
